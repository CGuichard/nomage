## Getting started

### Installation

Install `nomage` with pip:

```bash
pip install nomage
```

Install `nomage` with pip from source:

```bash
pip install git+https://github.com/CGuichard/nomage.git
# pip install git+https://github.com/CGuichard/nomage.git@<tag>
```
